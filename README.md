# Задача:
```
Определить структурный тип, описывающий расписание полетов самолетов
(пункт посадки, время отправления, время прибытия, время полета, стоимость билета, тип рейса: чартер, транзит, стыковка).
Заполнить структурный массив 20-ю записями.
1. Переписать из исходного массива в другой массив, информацию только о тех рейсах
   пункт назначения, которых оканчивается сочетанием «инск» и рейс стыковочный.
2. Затем новый массив отсортировать по времени полета (рационально переставлять
   все поля структуры разом).
3. Вывести все данные по конкретному рейсу.
4. Вывести 2 чартерных рейса, которые вылетают утром.
5. Реализовать функцию изменения рейса.
6. В отдельный массив поместить все чартерные рейсы, стоимостью более N рублей.
Реализовать вывод отфильтрованных данных в виде оберточной функции.
```

# Функции

```.cpp
string encode(string s) { // Меняю пробел на _ чтобы корректно записать в тхт файл
	for (char& c : s)
		if (c == ' ') c = '_';
	return s;
}

string decode(string s) { Меняю _ обратно на пробел при прочтении файла
	for (char& c : s)
		if (c == '_') c = ' ';
	return s;
}

void writeToTxt(ScheduleAirline* arr, int size) { // Забираю данные из массива и заношу в файл
	std::ofstream fout("file.txt");

	for (int i = 0; i < size; i++) {
		fout << encode(arr[i].pickup_point) << " "
			<< arr[i].departure_time.hours << " "
			<< arr[i].departure_time.minutes << " "
			<< arr[i].arrival_time.hours << " "
			<< arr[i].arrival_time.minutes << " "
			<< arr[i].flight_time << " "
			<< arr[i].ticket_price << " "
			<< (int)arr[i].flight_type << "\n";
	}

	fout.close();
}


void readFromTxt(ScheduleAirline* arr, int size) { // заполняю массив из данных txt
	std::ifstream fin("file.txt");

	for (int i = 0; i < size; i++) {
		fin >> arr[i].pickup_point
			>> arr[i].departure_time.hours
			>> arr[i].departure_time.minutes
			>> arr[i].arrival_time.hours
			>> arr[i].arrival_time.minutes
			>> arr[i].flight_time
			>> arr[i].ticket_price;

		arr[i].pickup_point = decode(arr[i].pickup_point);
		int type;
		fin >> type;
		arr[i].flight_type = (Flight_type)type;
	}

	fin.close();
}
```
